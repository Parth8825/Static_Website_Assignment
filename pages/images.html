      Windows (created by Microsoft) is the most popular and dominant OS on todays market. All over the world, around 90 % of people are using windows for their businesses or study. The two most popular versions of Windows for the desktop were XP and 7. Currently Windows 10 being the latest version and providing so many new features to attract many people. Most of the popular manufacture companies are making their hardware compatible with Windows which makes windows more smooth in almost all kinds of new hardware.

      Linux also known as GNU/Linux is one and only one operating system as an open source for computer devices, which means that typically we can modify all the codes in it and used it to make proper suitable for the hardware. The name “Linux” comes from the Linux kernel, which is started in 1991 by Linus Torvalds. The system’s utilities and libraries usually comes from the GNU operating system.

      Mac OS x is the operating system that is created by Apple company. This OS is only available for Mac Pcs and laptops. Unlike predecessor referred to classis or OS 10, OS x is the UNIX based operating system. Currently, this OS is considered as the fastest OS in the earth and on the second number stands Windows. The fun fact is that, Apple has chosen the name to each version of OS is inspired by the cat with 10.0 being Cheetah, Puma, Jaguar and all other wild cats.

      Android is based on Linux therefore, it is an open source operating system, but this OS is only limited for mobile device. It is developed by the Open Handset Alliance led by Google. After that, Google had acquired the Android Inc. and help from the Alliance to modify certain telecommunication system to enter in mobile ecosystem. Right now, Android is an number one operating system and has a 85% popularity and usage from all over the world.
